# Everon

[English](README.md) | [Русский](README.ru.md)

Everon is a lightweight Windows tray app that helps keep your computer awake when you need it.  
It is simple to use, runs quietly in the background, and gives quick access to all controls from the system tray.

## What Everon Can Do

- Prevent automatic sleep while enabled
- Optionally keep the display on
- Optionally send periodic key presses (F15/F16/F17)
- Work with timer modes:
  - Indefinite
  - For a selected duration
  - Until a selected time of day
- Toggle on/off from tray menu
- Open settings and configure behavior in a simple dialog
- Support global hotkey for quick toggle
- Start with Windows (optional)
- Show notifications for important events
- Support multiple interface languages

## Typical Use Cases

- Long downloads or uploads
- Presentations and meetings
- Monitoring tasks
- Keeping a workstation active during short breaks

## Quick Start

1. Run Everon.
2. Find its icon in the Windows system tray.
3. Right-click the icon to open the menu.
4. Open **Settings** and choose your preferred options.
5. Enable or disable Everon anytime from the tray menu or hotkey.
