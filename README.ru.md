# Everon

[English](README.md) | [Русский](README.ru.md)

Everon — это легкое приложение для Windows, которое помогает не давать компьютеру уходить в сон, когда это нужно.  
Оно работает в системном трее, не мешает работе и позволяет быстро управлять режимом активности.

## Что умеет Everon

- Предотвращать автоматический сон системы
- По желанию удерживать включенным экран
- По желанию отправлять периодические нажатия клавиш (F15/F16/F17)
- Работать в режимах таймера:
  - Бесконечно
  - На выбранную длительность
  - До выбранного времени
- Включаться и выключаться из меню в трее
- Открывать настройки в простом диалоговом окне
- Поддерживать глобальную горячую клавишу для быстрого переключения
- Запускаться вместе с Windows (опционально)
- Показывать уведомления о важных событиях
- Поддерживать несколько языков интерфейса

## Для каких задач подходит

- Длительные загрузки и выгрузки
- Презентации и встречи
- Мониторинг процессов
- Поддержание активности рабочего ПК во время коротких пауз

## Быстрый старт

1. Запустите Everon.
2. Найдите иконку приложения в системном трее Windows.
3. Нажмите правой кнопкой по иконке, чтобы открыть меню.
4. Откройте **Settings** и выберите нужные параметры.
5. Включайте и отключайте Everon через меню в трее или горячую клавишу.
